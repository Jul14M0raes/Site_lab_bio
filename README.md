# Site_lab_bio

RESUMO
O laboratório de BioGeoQuímica da UNIR é focado em investigar e fornecer subsídios técnico científicos para a comunidade acadêmica como também para a sociedade. Atualmente possue uma página web com domínio próprio (-unir.org).Apesar de tamanha autonomia foi observado que o site atual versão 'janeiro 2026' possuí uma interface não muito adequada onde se cabe atualizações e melhorias, destacando a padronização e implementação de identidade visual profissional no laboratório.
